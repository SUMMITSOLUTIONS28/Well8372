GOPHISH INSTALLATION

Open-Source Phishing Simulation Toolkit

ETHICAL USE WARNING

Gophish is designed for legitimate security awareness training. Use only on systems you own or have explicit permission to test. Unauthorized phishing attacks are illegal and unethical.

Installation Guide

Update System

security@kali:~
sudo apt update && sudo apt upgrade -y

Install Git

security@kali:~
sudo apt install git -y

Clone Gophish

security@kali:~
git clone https://github.com/gophish/gophish.git

Enter Directory

security@kali:~
cd gophish

Run Gophish

security@kali:~/gophish$
./gophish
