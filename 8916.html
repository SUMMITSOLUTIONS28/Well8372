Email Spoofing:

- Tool: sendemail (Linux) or swaks (Linux)

- Procedure:

1. Install swaks

sudo apt-get install swaks

2. Spoof email using swaks:

swaks --to victim@example.com --from fake@example.net --header "Subject: Test" --body "Spoofed email" --server smtp.example.com

(Replace with target email, fake sender, and SMTP server)
