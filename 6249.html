Security Tools Overview

HYDRA

How it works: Tries username/password combos via brute-force

Common uses:

  • Brute-force login pages (HTTP, FTP, SSH, etc.)
  • Test password strength

Example: hydra -l admin -P rockyou.txt http://site.com/login

Results:

[22][SSH] host: admin password123

Failure messages if no valid combos found

JOHN THE RIPPER

How it works: Cracks password hashes using wordlists or brute-force

Common uses:

  • Crack password hashes (e.g., from leaked DBs)
  • Test password strength

Example: john --wordlist=rockyou.txt hashes.txt

Results:

admin:password123 (hashes.txt)

Stats on cracked passwords

OWASP ZAP FUZZING

How it works: Sends malformed data to web app inputs

Common uses:

  • Test input validation
  • Find vulnerabilities (e.g., XSS, SQLi)

Example: Use ZAP's Fuzzing tool on a login form

Results:

Highlighted vulnerabilities in ZAP's interface

Payload responses (e.g., error messages, XSS triggers)

MEDUSA

How it works: Brute-force login attempts (like Hydra)

Common uses:

  • Test password strength on various services
  • Faster than Hydra in some cases

Example: medusa -h site.com -u admin -P rockyou.txt -M http

Results:

ACCOUNT FOUND: admin password (http://site.com)

Failure messages if no valid combos found
